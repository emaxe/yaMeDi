## 5. Должностные обязанности и регламент работ

### Действия директолога в начале месяца:
1. Получить от аккаунт-менеджера согласованный план на месяц.
2. Проверить плановые показатели в отчете.
3. Ознакомиться с прогнозом и приоритетными задачами на месяц.
4. Подготовить детальный план работ для достижения KPI / дать комментарии по задачам руководителя:
   * Оптимизация текущих кампаний;
   * Запуск новых гипотез;
   * Перераспределение бюджета;
   * Корректировка стратегий.
5. Зафиксировать ключевые гипотезы на месяц в листе «Гипотезы».

### Ответственность аккаунт-менеджера:
1. Согласовать план с клиентом.
2. Внести плановые показатели в отчет:
   * Месячный лист;
   * Недельный лист.
3. Уведомить директолога об утвержденном плане и изменениях по проекту.

### Обязанности руководителя:
1. Подготавливает данные с прогнозом каждый месяц.
2. Проводит аудит рекламных кампаний — 1 раз в месяц.
3. Раз в три месяца созванивается с менеджером из Яндекса и согласовывает задачи (совместно с аккаунтом / директологом).
